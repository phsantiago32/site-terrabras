# Terrabrás Já 🇧🇷

Este é o repositório do movimento **Terrabrás Já**, uma landing page/manifesto desenvolvida para alertar a população brasileira sobre a exploração estrangeira das nossas reservas de Terras Raras e exigir a criação de uma estatal forte e soberana para gerir essa riqueza: a Terrabrás S.A.

## 🎯 Objetivo

O site tem o formato de "dossiê" e landing page de alta conversão, projetado para:
- Informar sobre os perigos da privatização e exportação predatória das Terras Raras (minerais críticos da tecnologia moderna).
- Denunciar políticos e lobbies entreguistas.
- Coletar assinaturas na petição oficial e incentivar o compartilhamento (efeito viral).
- Funcionar perfeitamente no formato mobile (para facilitar leitura estilo "stories" e a ação de "tirar print" e compartilhar nas redes sociais).

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando um ecossistema moderno voltado para performance, acessibilidade e design de alto impacto:

- **React 18**: Biblioteca JavaScript para construção da interface.
- **Vite**: Ferramenta de build rápida e eficiente.
- **TypeScript**: Tipagem estática para maior segurança e facilidade de manutenção.
- **Tailwind CSS**: Framework de CSS utilitário para estilização rápida e responsiva.
- **Motion (Framer Motion)**: Biblioteca de animações para React, garantindo transições suaves.
- **Lucide React**: Biblioteca de ícones (elegante, personalizável e leve).

## 🚀 Como Executar o Projeto Localmente

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

1. **Instale as dependências:**
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Acesse a aplicação:**
   Abra o seu navegador e acesse a URL local (geralmente `http://localhost:3000`).

## 📦 Scripts Disponíveis

No diretório do projeto, você pode rodar os seguintes comandos:

- `npm run dev` - Roda o app em modo de desenvolvimento.
- `npm run build` - Cria o pacote de produção na pasta `dist`.
- `npm run preview` - Pré-visualiza localmente a aplicação gerada pelo comando de build.
- `npm run lint` - Executa a verificação de linter nos arquivos do projeto para garantir a qualidade do código.

## 🎨 Estrutura e Design

A aplicação adota um visual que reflete a urgência e seriedade do movimento: cores que remetem à riqueza do subsolo (tons de cobre e amarelo), ao verde do Brasil e um fundo escuro que gera alto contraste, perfeito para a tipografia forte (estilo de "cartaz" e manifesto).

As animações scroll e backgrounds auxiliam a direcionar o usuário para a principal Call To Action (Assinar no Change.org / Compartilhar a Página).
